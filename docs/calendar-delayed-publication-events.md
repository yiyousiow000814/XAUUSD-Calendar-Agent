# Economic Calendar Delayed/Update-Slot Events

This repo exports economic calendar entries under `data/Economic_Calendar`.

Some providers publish the same event twice within a short time window:
1. First row: placeholder (Actual missing; Forecast/Previous may exist).
2. Second row: updated (Actual filled), sometimes at a slightly different time.

We call these "update-slot duplicates".

## Detection Heuristic

1. Same `Cur.` and `Event`.
2. Timestamps within 60 minutes (including cross-midnight).
3. Forecast/Previous are compatible (if both sides have a value, they match).
4. Exactly one row has an `Actual` value.
